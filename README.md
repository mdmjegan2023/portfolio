## 🚀 Getting Started

### Prerequisites
- Node.js >= 18.x
- npm

### Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/mdmjegan2023/jegannathan-portfolio.git

# 2. Navigate into the project
cd jegannathan-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder.

## ☁️ Deploy on Vercel

This project is configured to deploy automatically on Vercel.

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) → Import your GitHub repo
3. Vercel auto-detects Vite — just click **Deploy**
4. Your site will be live in seconds ✅

> No extra configuration needed. Vercel handles `npm install` and `npm run build` automatically.
